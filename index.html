How it reads the image

01

Frame it

The image is scanned for what's actually depicted — subjects, action, layout — the way an editor would describe it over the phone.

02

Weigh the context

A product photo, a chart, and a decorative flourish all need different alt text. Context and length settings shape which one you get.

03

Write it plainly

No "image of," no guessing at emotion, no keyword stuffing — just the information a sighted user would get at a glance.

Generated alt text is a strong first draft — always give it a read before it ships. Nothing you upload is stored.
